# IfBest Enterprise — Корпоративная видеотека

Корпоративная видеотека для компании IfBest Enterprise.

## Текущий этап

Проектирование базы данных (Функциональное требование №1).

## Стек

- **СУБД:** PostgreSQL 16
- **ORM:** SQLAlchemy 2.0
- **Миграции:** Alembic
- **Инфраструктура:** Docker Compose
- **Язык:** Python 3.11+
